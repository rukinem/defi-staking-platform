# 💰 defi-staking-platform

Decentralized staking platform with reward distribution and auto-compounding

## Features

- Stake ERC-20 tokens
- Auto-compound rewards
- Flexible lock periods
- Reward distribution
- Emergency withdraw

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
